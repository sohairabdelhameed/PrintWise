# PrintWise App 🖨️

PrintWise is a platform that simplifies the process of getting documents printed. Users can upload their documents, specify printing preferences, and receive quotes from various print shops. The app facilitates decision-making by providing users with information on the best price and quality for their printing needs.

## Key Features 🌟

- **Document Upload:** Users can easily upload documents they want to get printed, supporting various file formats.

- **Printing Preferences:** A user-friendly questionnaire guides users through selecting their printing preferences, such as color options, paper quality, and more.

- **Price and Quality Comparison:** The app calculates and displays quotes from different print shops based on user preferences, allowing users to compare prices and quality.

- **Shop Selection:** Users can choose the print shop that best fits their requirements, whether it's the best price or the highest quality.

- **Order Submission:** Once a decision is made, the user's document and printing preferences are sent to the selected print shop for processing.

## Getting Started 🚀

To try out PrintWise, follow these steps:

1. Clone the repository: `git clone https://github.com/your-username/printwise.git`
2. Navigate to the project directory: `cd printwise`
3. Install dependencies: `npm install`
4. Set up your Firebase project and configure the environment variables.
5. Run the application: `ng serve`
6. Open your browser and visit `http://localhost:4200/` to start exploring the streamlined document printing process.

## Technologies Used 💻

- **Frontend:** [Angular](https://angular.io/) 🅰️
- **Backend:** [Firebase](https://firebase.google.com/) 🔥
- **Authentication:** Firebase Authentication for secure user authentication.
- **Database:** Firebase Firestore for storing user data and printing preferences 🗃️.
